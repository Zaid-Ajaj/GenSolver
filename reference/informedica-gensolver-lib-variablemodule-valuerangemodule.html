TypeDescription
ValueRange

ValueRange represents a discrete set of non-zero positive rational numbers, the set is either finite and then it is a set or it is infinite and then it is a range.

Functions to handle Value A Value is a non zero positive value implemented as a BigRational. Basic arrhythmic operations can be performed with this type.
