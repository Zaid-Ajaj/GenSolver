Range

A Range is an unlimited set of rational numbers, when a set has both a minimum, maximum and an increment then it is not a range anymore but a list of values
Values

Values is a discrete set of non-zero positive rational numbers, the set is either limited and then it is a list or it is unlimited and then it is a range.
